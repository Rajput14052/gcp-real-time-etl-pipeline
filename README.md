# GCP Real-Time ETL Pipeline

## Overview
This project demonstrates a real-time ETL pipeline built using GCP.

## Problem
Batch processing caused delay in data availability.

## Solution
Designed a real-time pipeline using Pub/Sub, Dataflow, and BigQuery.

## Architecture
Source → Pub/Sub → Cloud Function → Dataflow → BigQuery → Power BI

## Tech Stack
Python, GCP, BigQuery, Dataflow, Pub/Sub, Airflow, pytest

## Features
- Real-time ingestion
- Data transformation
- Data quality checks
- Pipeline orchestration

## Impact
- Reduced manual effort by 60%
- Improved performance by 40%
- Reduced cost by 35%
